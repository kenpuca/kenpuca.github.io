! The topic of this course is the theory and practice of programming languages. We will be focused on the design principles and programming patterns of several programming languages from different programming paradigms. We will pay particular attention to the program techniques in functional programming, and why they are gaining greater importance in modern programming.

Programming Languages

Course Outline

This course is divided into three parts.

  1. Foundations of programming languages
  2. Practical languages
  3. Survey


(I) Foundations


!

Computation

  • Stateful computation
  • Functional computation


!

Design of programming languages

  • Organization of code
  • Operational semantics
  • Type system
  • Denotational semantics


!

Functional programming

  • Introduction to Clojure
  • Programming without mutable data


!

Programming Paradigms

  • Procedural programming
  • Object-oriented programming
  • Pure functional programming
  • Programming in reality


(II) Practical languages


!

Scala

  • Pure object orientation in Scala
  • Strong type system
  • Functional programming in Scala
  • Concurrency


!

Clojure

  • Clojure as Lisp
  • Functional programming in Clojure
  • Object-oriented programming in Clojure
  • Concurrency
  • Macros


(III) Survey


!

  • More languages (e.g. Haskell, Javascript/Typescript, Go)
  • More paradigms (e.g. Logical programming with Prolog)
